## first install this packages in windows
pyfirmata
~~~
pip install pyfirmata
~~~
playsound
~~~
pip install playsound
~~~
gtts
~~~
pip install gtts
~~~
speechrecognition
~~~
pip install SpeechRecognition
~~~
## The parts used in this project are:
- SG90 Servo Motor *3
- Ultrasonic module
- LM35
- SSR Relay
- plastic case
- DC Power Supply
